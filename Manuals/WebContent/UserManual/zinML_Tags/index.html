zint Markup Language (zintML)

index page

Introduction

There are three types of zintML commands.

  • single line such as //
  • 2-line such as R
  • 3-line such as 2

Syntax

A zintML markup starts a

  1. # at column 1
  2. one or more command codes starting column 2
  3. if it is 2-line
    1. V indicating the beginning a block (as if an arrow downwards)
    2. A indicating the end of the block (as if an arrow upwards)
  4. if it is 3-line
    1. V indicating the beginning the first block (as if an arrow downwards)
    2. M indicating the beginning the second block (comes from "Middle")
    3. A indicating the end of the block (as if an arrow upwards)

Examples

1-line

    #// comment is 1-line

2-line

    #RV Remark is 2-line
    aaa
    #RA Remark is 2-line

3-line

    #2V 2-column is 3-line
    aaa
    #2M 2-column is 3-line
    bbb
    #2A 2-column is 3-line
